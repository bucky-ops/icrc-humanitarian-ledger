# ICRC Humanitarian Blockchain (Phase 1, 2 & 3)
A bare-metal Node.js implementation of a supply chain ledger.

## Current Progress:
- [x] Phase 1: Folder Structure Established
- [x] Phase 1: Cryptographic Identity Module (secp256k1)
- [x] Phase 1: MedicalKit Asset Model with Self-Signing Logic
- [x] Phase 1: Local Environment Verification
- [x] Phase 2: Blockchain Manager with File-Based Storage
- [x] Phase 2: Validation Logic for Medical Kits
- [x] Phase 2: Chain Integrity Verification
- [x] Phase 3: API Gateway with Express.js
- [x] Phase 3: RESTful Endpoints for Ledger Access
- [x] Phase 3: Kit Submission and Validation API

## Security Note:
Private keys are excluded from version control via .gitignore.
